AI-Powered Sentiment Analysis Tool

Smart Customer Review Analytics Platform

Project Description

This project is developed using Artificial Intelligence, Machine Learning, and Natural Language Processing techniques to analyze customer reviews automatically. The system predicts whether reviews are positive, negative, or neutral and provides analytics through dashboards and visualizations.

Main Features

  • ✅ Sentiment Prediction
  • ✅ Emotion Detection
  • ✅ Dashboard Analytics
  • ✅ CSV Bulk Upload
  • ✅ Word Cloud Visualization
  • ✅ Review History Storage
  • ✅ AI-Based Review Analytics
  • ✅ Login Authentication System

Technologies Used

  • 🐍 Python
  • 🌐 Flask
  • 🤖 Machine Learning
  • 🧠 Natural Language Processing
  • 📊 Matplotlib
  • ☁️ WordCloud
  • 💾 SQLite Database
