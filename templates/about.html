AI-Powered Sentiment Analysis Tool

Project Description

This project is developed using Machine Learning and Natural Language Processing. The system analyzes user reviews and predicts whether the sentiment is: Positive, Negative, or Neutral.

Main Features

  • Sentiment Prediction
  • Emotion Detection
  • CSV Bulk Upload
  • Word Cloud Visualization
  • Pie Chart Analytics
  • History Storage
  • Analytics Dashboard
  • User Login Authentication

Technologies Used

  • Python
  • Flask
  • Machine Learning
  • Natural Language Processing
  • SQLite Database
  • Matplotlib
  • WordCloud
